Para startar o projeto em sua máquina, faça um clone do repositório e execute os comandos:

```
yarn && yarn start
```

ou

```
npm install && npm run start
```

Libs utilizadas.

- Styled components
- redux/toolkit
- prop-types
- redux-persist
- uuid
- axios
- eslint
- prettier
